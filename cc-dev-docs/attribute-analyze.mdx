Understanding which creators drive results is essential for optimizing your partnerships. This guide covers attribution tracking and performance analysis.

## How Attribution Works

### Cart Attributes

**Automatic Tracking:**
When customers click creator links or visit creator storefronts, CreatorCommerce sets cart attributes:
- `cc-creator-id` - Creator's unique ID
- `cc-creator-handle` - Creator's handle (for metaobject lookup)
- `cc-campaign-id` - Campaign ID (if applicable)

**These attributes persist:**
- Through checkout process
- In order metadata
- For attribution tracking
- Across session

### Order Attribution

**Attributed Orders Include:**
- Creator ID in order metadata
- Creator handle in order attributes
- Campaign ID (if applicable)
- Discount code used (creator-specific)

**What Gets Tracked:**
- Order value
- Product purchases
- Customer information
- Conversion path

## Attribution Methods

### Last-Click Attribution

**How It Works:**
- Credits sale to last creator link clicked
- Simple, straightforward
- Most common method
- Easy to implement

**Best For:**
- Direct creator traffic
- Clear conversion paths
- Simple attribution needs

### First-Click Attribution

**How It Works:**
- Credits sale to first creator interaction
- Shows awareness impact
- Values discovery
- Long-term view

**Best For:**
- Multi-touch campaigns
- Understanding awareness
- Measuring brand building

### Multi-Touch Attribution

**How It Works:**
- Credits multiple creators in path
- Shows full customer journey
- Complex but accurate
- Requires additional setup

**Best For:**
- Complex customer journeys
- Multiple creator touchpoints
- Advanced analytics needs
