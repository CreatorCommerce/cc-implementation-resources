# Connect Your Tech Stack

CreatorCommerce integrates with your existing tech stack to work seamlessly alongside your current tools. This guide covers the key integration points.

## Core Integrations

### Shopify (Required)

**Native Integration:**
- CreatorCommerce is a Shopify app
- Installs directly in your Shopify store
- Uses Shopify metaobjects for data storage
- Leverages Shopify collections for drops

**What It Does:**
- Syncs creator data to Shopify metaobjects
- Maps drops to Shopify collections
- Stores creator profiles and content
- Enables Liquid template access to creator data

**No Additional Setup Required:**
- Metaobjects auto-created on app install
- Collections auto-synced with drops
- Cart attributes flow through checkout

### Shopify Collabs (Recommended)

**Integrated Creator Management:**
- Seamless creator onboarding
- Unified creator database
- Automatic discount code sync
- Integrated tier management

**Benefits:**
- Single source of truth for creators
- Reduced manual work
- Better creator experience
- Simplified creator lifecycle

**Setup:**
- Install Shopify Collabs app
- Configure creator tiers
- Enable CreatorCommerce integration
- Creators sync automatically
