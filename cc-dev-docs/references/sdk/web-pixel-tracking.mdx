CreatorCommerce web pixel tracking complements SDK cart attributes to provide funnel analytics and attribution quality.

## What the Web Pixel Does

- Tracks funnel behavior across creator-context traffic
- Supports attribution analysis and reporting
- Works with cart attributes and discount context

## Recommended Setup

1. Enable the CreatorCommerce SDK app embed.
2. Enable the CreatorCommerce web pixel.
3. Verify cart attributes and pixel events on creator-link sessions.
