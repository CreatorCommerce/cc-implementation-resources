## Available Actions

CreatorCommerce exposes actions in Shopify Flow that let you automate partner program operations without writing code.

| Action | What It Does | Common Use |
|--------|-------------|-----------|
| **Sync Tier Assignment with CreatorCommerce** | Syncs creator and tier data into CreatorCommerce (commonly from Shopify Collabs) | Create/update collabs from Flow variables |

## Action Inputs (Currently Available)

For **Sync Tier Assignment with CreatorCommerce**, these input fields are currently available:

- `First Name`
- `Last Name`
- `Email`
- `Location`
- `Tier Name`
- `Affiliate Code`
- `Community Member Id`
- `Primary Username`
- `MyShopify Domain`

<Note>
This action is often used with Shopify Collabs variables, but it can also be used with custom data sources as long as the required values are mapped.
</Note>

## Using Actions in Flow

### Sync Collab from Shopify Collabs

```
Trigger: Shopify Collabs event (or another Flow trigger)
Action: Sync Tier Assignment with CreatorCommerce
  - First Name
  - Last Name
  - Email
  - Tier Name
  - Affiliate Code
  - Community Member Id
  - Primary Username
  - MyShopify Domain
```

### Sync Custom Source Data

```
Trigger: Any Shopify Flow trigger with creator data available
Action: Sync Tier Assignment with CreatorCommerce
```

## Combining Triggers and Actions

The real power is combining CC triggers with CC actions and other Shopify Flow connectors:

| Trigger | + Condition | + Action |
|---------|------------|----------|
| Shopify Collabs / custom trigger | Creator data present | Sync Tier Assignment with CreatorCommerce |
| CreatorCommerce - Creator Updated | Affiliate code changed | Sync to external CRM |
| CreatorCommerce - Creator Updated | Shop handle changed | Update downstream links |
