## Consumer Co-branded Events (B2C)

These customer-facing co-branded events are sent by Shopify Flow templates using Klaviyo `Track an Event`.

| Shopify Flow Template | Event Name in Klaviyo | Trigger |
|-----------------------|-----------------------|---------|
| `CC // Log a customer abandoned cart Klaviyo event` | `Creator Checkout Abandoned` | Customer abandons checkout |
| `CC // Log a custom post-purchase Klaviyo event` | `Creator Attributed Order` | Order created |
| `CC // Log a customer marketing opt-in Klaviyo event` | `Marketing Opt In From Creator` | Customer marketing opt-in/subscription event |

## B2C Track an Event Payload Mapping

Each template maps the following into the Klaviyo action:

- `Event Properties` -> `{{getMetaobjectEntry.data}}`
- `Customer Properties` -> creator/collab dictionary from `getMetaobjectEntry.*`

Typical `Customer Properties` payload:

```json
{
  "cc-creator-first-name": "{{getMetaobjectEntry.ccCreatorFirstName}}",
  "cc-creator-last-name": "{{getMetaobjectEntry.ccCreatorLastName}}",
  "cc-creator-profile-picture": "{{getMetaobjectEntry.ccCreatorProfilePicture}}",
  "cc-creator-shop-title": "{{getMetaobjectEntry.ccCreatorShopTitle}}",
  "cc-creator-shop-theme-primary-color": "{{getMetaobjectEntry.ccCreatorShopThemePrimaryColor}}",
  "cc-creator-shop-handle": "{{getMetaobjectEntry.ccCreatorShopHandle}}",
  "cc-collab-affiliate-link": "{{getMetaobjectEntry.ccCollabAffiliateLink}}",
  "cc-collab-discount-amount": "{{getMetaobjectEntry.ccCollabDiscountAmount}}",
  "cc-collab-discount-code": "{{getMetaobjectEntry.ccCollabDiscountCode}}",
  "cc-collab-affiliate-code": "{{getMetaobjectEntry.ccCollabAffiliateCode}}",
  "cc-collab-discount-type": "{{getMetaobjectEntry.ccCollabDiscountType}}",
  "cc-creator-shop-description": "{{getMetaobjectEntry.ccCreatorShopDescription}}"
}
```

<Note>
Some templates include additional keys (for example first/last name fields are often included for checkout abandoned and post-purchase flows). Keep your Klaviyo template logic resilient to missing optional fields.
</Note>

## Klaviyo Template Access Syntax

- Profile properties: `{{ person|lookup:"field-name" }}`
- Event properties: `{{ event.property_name }}`
- Nested event properties: `{{ event.property.nested_variable }}`

## B2C Flow Trigger Examples

### Cart Abandonment with Creator Context

```
Trigger: Creator Checkout Abandoned
Wait: 2-4 hours
Filter: Has not placed order
Action: Send co-branded recovery email
```

### Post-Purchase with Creator Context

```
Trigger: Creator Attributed Order
Action: Send co-branded thank you / follow-up flow
```

### Opt-In Welcome with Creator Context

```
Trigger: Marketing Opt In From Creator
Action: Enter creator-personalized welcome/nurture branch
```

## Segmentation

Create segments based on CC properties:

- **CC Referred Customers**: `cc-creator-shop-handle is set`
- **High-Value Creator Traffic**: `cc-creator-shop-handle is set AND Total CLV > 200`
- **Specific Creator Audience**: `cc-creator-shop-handle equals sarah-jones`
