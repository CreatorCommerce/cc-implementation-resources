Use this as a blueprint for shaping co-branded funnels by brand category. Each section maps funnel stages to the UI components in the **Co‑branded Storefront UI Glossary**, with category‑specific additions highlighted.

## How to Read This

- **Stages** map the shopper journey from entry to checkout.
- **Core components** are universal ecommerce blocks.
- **Co‑branded layers** are creator/affiliate‑specific modules.
- **Category add‑ons** are unique to the vertical.

---

## Fashion & Apparel

**Funnel Goal:** Rapid inspiration → fit confidence → outfit completion.

### Landing / Entry

- **Core components:** `Hero banner`, `Category grid`, `Featured collection`, `UGC gallery`.
- **Co‑branded layers:** `Creator header`, `Co‑brand lockup`, `Creator picks rail`, `Creator quote callout`.
- **Category add‑ons:** `Complete the look`, `Styling tips`.

### Collection / Browse

- **Core components:** `Product grid`, `Filters`, `Swatches`, `Quick add`.
- **Co‑branded layers:** `Creator tag filter`, `Creator recommendation badges`.
- **Category add‑ons:** `Fit notes`, `Size matrix`.

### Product Detail (PDP)

- **Core components:** `Product media gallery`, `Variant selector`, `Size guide`, `Reviews summary`.
- **Co‑branded layers:** `Creator review block`, `Creator UGC gallery`, `Creator tips`.
- **Category add‑ons:** `Fit recommender`, `Complete the look`.

### Cart / Checkout

- **Core components:** `Cart upsell`, `Progress bar`, `Order summary`.
- **Co‑branded layers:** `Creator cart banner`, `Attribution summary`, `Creator thank‑you block`.
- **Category add‑ons:** `Bundle add-on` (look‑based bundles).

## Checklist for Any Category

- Ensure **creator context** is visible at entry and persists through cart.
- Add **creator‑specific proof** (quotes, UGC, reviews) on PDPs.
- Use **category add‑ons** to close the confidence gap (fit, shade, regimen, room).
- Validate **creator absent** fallback for every module.
