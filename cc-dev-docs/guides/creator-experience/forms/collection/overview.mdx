Collection forms gather specific content from partners after they've onboarded. Use these for ongoing content collection.

## Use Cases

| Form | Collects | When to Use |
|------|----------|-------------|
| Bio update | New or updated bio | Profile refresh campaigns |
| Social content | UGC, testimonials | Content campaigns |
| Seasonal | Holiday picks, summer favorites | Seasonal marketing |
| Testimonial | Product reviews, quotes | Social proof collection |

## Form Types

### Bio Collection

```
Fields:
- Short bio (2-3 sentences)
- Long bio (paragraph)
- Tagline
- "Why I love [Brand]"
```

### Social Content

```
Fields:
- Image upload
- Video URL
- Caption/description
- Usage rights checkbox
```

### Testimonial

```
Fields:
- Product select (dropdown)
- Rating (1-5)
- Written testimonial
- Photo with product (optional)
```

## Triggering Collection

### Email Campaigns

Send collection requests via email:
- Subject: "Share your summer favorites"
- Magic link to form
- Deadline for submissions

### Dashboard Prompts

Show in-dashboard prompts:
- "Your bio is empty—add one now"
- "We're collecting holiday content"

### Automated Triggers

Workflow-triggered collection:
- 30 days since last update
- New product launch
- Milestone reached

## Progressive Collection

Don't ask for everything at once:

```
Week 1: Onboarding basics
Week 2: Bio and photo
Week 4: Product picks
Week 8: Testimonial request
```

## Content Moderation

Options for reviewing submitted content:
- **Auto-publish:** Goes live immediately
- **Queue for review:** Admin approval required
- **AI screening:** Automated quality check
