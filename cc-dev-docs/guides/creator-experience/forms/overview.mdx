CC Forms & Dashboard takes authentication, infrastructure, and design off your shoulders. You get a fully-functional partner portal without building one.

## What You Get

### Authentication Handled
- Magic link emails
- SSO integration
- Session management
- Password reset flows

### Infrastructure Managed
- Hosting and uptime
- Data storage and security
- API rate limiting
- Backup and recovery

### Design Customizable
- CSS theming
- Logo and brand colors
- Custom fonts
- Layout options

## Form Types

| Form Type | Purpose | When to Use |
|-----------|---------|-------------|
| Onboarding | Initial partner signup and profile | First interaction |
| Collection | Gather specific content (bio, social) | After signup |
| Product | Product/collection selection | Curating drops |
| Custom | Any structured data | Unique requirements |

## Quick Start

1. **Choose form type** based on what you're collecting
2. **Configure fields** in CC admin
3. **Generate link** with optional SSO
4. **Share with partners** via email, embed, or direct link
5. **Style with CSS** to match your brand
