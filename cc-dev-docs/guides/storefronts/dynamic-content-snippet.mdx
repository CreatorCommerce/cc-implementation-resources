## The Challenge

You want section settings like headlines to support both:
- Static copy: "Shop our best sellers"
- Dynamic copy: "Shop [CREATOR_NAME]'s picks"

## The CC Dynamic Text Snippet

Create `snippets/cc-dynamic-text.liquid`:

```liquid
{% comment %}
  Usage: {% render 'cc-dynamic-text', text: section.settings.heading, cc_creator: cc_creator %}
{% endcomment %}

{% liquid
  assign output = text
  
  if cc_creator
    assign output = output | replace: '[CREATOR_NAME]', cc_creator.cc-creator-first-name
    assign output = output | replace: '[CREATOR_FULL_NAME]', cc_creator.cc-creator-first-name | append: ' ' | append: cc_creator.cc-creator-last-name
    assign output = output | replace: '[DISCOUNT_CODE]', cc_creator.cc-collab-discount-code
    assign output = output | replace: '[DISCOUNT_AMOUNT]', cc_creator.cc-collab-discount-amount
  else
    assign output = output | replace: '[CREATOR_NAME]', 'our expert'
    assign output = output | replace: '[CREATOR_FULL_NAME]', 'our team'
    assign output = output | replace: '[DISCOUNT_CODE]', ''
    assign output = output | replace: '[DISCOUNT_AMOUNT]', ''
  endif
%}

{{ output }}
```

## Usage in Sections

```liquid
<h1>
  {% render 'cc-dynamic-text', text: section.settings.heading, cc_creator: cc_creator %}
</h1>
```

## Supported Tokens

| Token | With Creator | Without Creator |
|-------|--------------|-----------------|
| `[CREATOR_NAME]` | Sarah | our expert |
| `[CREATOR_FULL_NAME]` | Sarah Jones | our team |
| `[DISCOUNT_CODE]` | SARAH15 | (empty) |
| `[DISCOUNT_AMOUNT]` | 15 | (empty) |

## Example Settings

In your section schema:

```json
{
  "type": "text",
  "id": "heading",
  "label": "Heading",
  "default": "Shop [CREATOR_NAME]'s favorites",
  "info": "Use [CREATOR_NAME], [DISCOUNT_CODE], [DISCOUNT_AMOUNT] for personalization"
}
```
