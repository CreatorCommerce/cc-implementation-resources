## What You'll Build

A landing page that displays:
- Creator's name, photo, and bio
- Their curated product picks
- Exclusive discount callout
- Branded call-to-action

## Prerequisites

- CreatorCommerce installed on your store
- At least one collab created
- Access to Shopify theme editor

## Quick Start

### 1. Create the Template

In Shopify Admin → Online Store → Themes → Edit code:

Create `templates/page.cc-creator.json` or use the metaobject template.

### 2. Add CC Sections

Add these sections to your template:
- CC Hero (creator intro)
- CC Product Grid (their picks)
- CC Banner (discount reminder)

### 3. Preview

Visit `/pages/creators/[handle]` to see the live page.
