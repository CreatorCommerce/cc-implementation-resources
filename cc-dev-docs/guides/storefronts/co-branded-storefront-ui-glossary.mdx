This glossary lists common ecommerce components plus co-branded patterns unique to creator/affiliate storefronts. Definitions are short so you can scan and map to your theme sections quickly.

## Global Navigation & Utility

- **Header** — Global navigation with logo, search, account, and cart.
- **Stateful header** — Header behavior changes by user/device/session state.
- **Announcement bar** — Top strip for promos, shipping, or alerts.
- **Auto-apply promo strip** — Header promo area communicating auto-applied discount incentives.
- **USP strip** — Short benefits row (shipping, returns, guarantee).
- **Mega menu** — Expanded nav with categories, promos, and images.
- **Decision-tree mega menu** — Hierarchical nav that routes by type/material/collection/occasion.
- **Tiered category labels** — High-level browse anchors like New, Featured, and Shop All.
- **Flyout menu** — Drawer-style navigation panel commonly used on mobile.
- **Nested nav panel** — Multi-level navigation inside a single menu system.
- **Menu image tile** — Visual category shortcut placed inside mega/flyout menus.
- **Breadcrumbs** — Location trail for page hierarchy.
- **Search bar** — Keyword input for catalog search.
- **Account entry** — Login and account access link.
- **Logged-in greeting** — Personalized account-state label for returning users.
- **Mini cart** — Slide-out cart summary.
- **Utility rail** — Persistent quick actions like search, wishlist, bag, and store locator.
- **Support widget** — Live chat or help CTA.
- **Find a store CTA** — Utility action linking to in-person store discovery.
- **Language/currency switcher** — Locale selection controls.
- **Trust badges** — Security, payments, or compliance marks.
- **Toast notifications** — Short messages for add-to-cart or errors.
- **Skeleton states** — Loading placeholders for async content.
- **Focus trap** — Accessibility behavior that keeps keyboard focus within active overlays.
