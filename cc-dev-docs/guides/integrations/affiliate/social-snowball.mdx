---
title: 'Social Snowball'
description: 'Connect Social Snowball affiliate programs with CreatorCommerce co-branded experiences.'
---

Sync Social Snowball affiliate data into CreatorCommerce so each affiliate can power a co-branded funnel experience in Shopify.

## Integration Type

- **Mode:** Webhook-based
- **Trigger source:** Social Snowball events posted to CreatorCommerce webhook endpoints
- **Shopify Flow required:** No (direct webhook pattern)

## What This Integration Does

| Source Integration Data | CreatorCommerce Outcome |
|-------------------------|------------------------|
| Affiliate profile and program events | Creators/collabs update for co-branded funnels |
| Program and discount context | Attribution context is preserved through CC redirects |

## Setup

1. Enable Social Snowball in CreatorCommerce.
2. Configure Social Snowball webhook delivery to CreatorCommerce for affiliate/program updates.
3. Use `CC //` naming on Social Snowball programs you want treated as co-branded programs.
4. Validate by updating a test affiliate and confirming creator/collab updates in CC.

## Webhook Endpoint (CC API)

Configure Social Snowball to POST to:

```http
POST /webhooks/integrations/ss/webhooks/program/changed
Content-Type: application/json
```
