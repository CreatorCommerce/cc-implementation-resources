Shopify Flow is the core automation layer for CreatorCommerce—trigger workflows on partner events and take actions across your Shopify store.

## What This Integration Does

| CC Triggers | CC Actions |
|-------------|------------|
| CreatorCommerce - Creator Updated | Sync Tier Assignment with CreatorCommerce |

## Native Integration

Shopify Flow is built into CC—no setup required. Just create flows using CC triggers and actions.

## Common Flows

### Sync Collab Data from Shopify Collabs
```
Trigger: Shopify Collabs event
Action: Sync Tier Assignment with CreatorCommerce
```

### React to Creator Updates
```
Trigger: CreatorCommerce - Creator Updated
Condition: affiliate code or shop handle changed
Action: Sync downstream systems (Slack / CRM / ESP / HTTP)
```
