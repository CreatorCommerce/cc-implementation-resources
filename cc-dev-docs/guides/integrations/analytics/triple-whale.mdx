Connect Triple Whale with CreatorCommerce for unified attribution across paid, organic, and partner channels.

## What This Integration Does

| CC → Triple Whale | Benefit |
|-------------------|---------|
| Partner attribution data | See partner performance alongside other channels |
| Co-branded page events | Track full funnel by partner |
| Order attribution | Unified conversion tracking |

## Use Cases

### Unified Attribution
See how partner traffic compares to paid ads, organic, and email in one dashboard.

### Partner CAC/LTV
Calculate customer acquisition cost and lifetime value by partner.

### Blended ROAS
Include partner-driven revenue in your overall ROAS calculations.

## Setup

<Steps>
  <Step title="Connect Triple Whale">
    In CC Admin → Integrations → Triple Whale, authenticate with OAuth.
  </Step>
  <Step title="Configure events">
    Choose which CC events to send to Triple Whale.
  </Step>
  <Step title="Map attribution">
    Configure how CC attribution maps to Triple Whale sources.
  </Step>
  <Step title="Verify">
    Check Triple Whale dashboard for incoming CC data.
  </Step>
</Steps>

## Events Sent

| Event | Data |
|-------|------|
| Page View | Partner handle, page type |
| Add to Cart | Partner attribution, products |
| Purchase | Full order data with partner attribution |

## Attribution Model

CC attribution integrates with Triple Whale's model:
- First-touch partner attribution
- Last-touch partner attribution
- Partner influence in multi-touch journeys
