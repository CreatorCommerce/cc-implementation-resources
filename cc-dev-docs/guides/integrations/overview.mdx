## Native Integrations

### Shopify

CreatorCommerce is built natively on Shopify:
- **Metaobjects** — Creator data stored in Shopify's native data layer
- **Liquid** — Templates render directly in your theme
- **Flow** — Triggers and actions for automation
- **Checkout** — Discount codes and attribution work seamlessly

### Klaviyo

Sync creator and customer data for co-branded email campaigns:
- Customer profiles enriched with partner attribution
- Creator profiles for partner communications
- Event triggers for automated flows

### Attentive

Enable co-branded SMS and MMS messaging:
- Partner context in subscriber profiles
- Dynamic message personalization
- Attribution tracking
