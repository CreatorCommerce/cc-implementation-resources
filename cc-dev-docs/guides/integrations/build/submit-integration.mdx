Built an integration with CreatorCommerce? Submit it for review to be featured in our integrations directory and made available to all CC users.

## Why Submit?

- **Visibility** — Featured in CC integrations directory
- **Distribution** — Available to all CC merchants
- **Support** — We help maintain compatibility
- **Partnership** — Potential co-marketing opportunities

## What We're Looking For

### Quality Integrations

- **Reliable** — Handles errors gracefully
- **Documented** — Clear setup instructions
- **Maintained** — Commitment to keep it working
- **Valuable** — Solves a real problem for merchants

### Integration Types

| Type | Examples |
|------|----------|
| Platform connectors | Affiliate networks, CRMs, CDPs |
| Analytics | Attribution, BI, reporting tools |
| Messaging | Email, SMS, push platforms |
| Content | DAM, UGC, content platforms |
| Automation | Workflow tools, iPaaS |

## Submission Process

<Steps>
  <Step title="Build and test">
    Create your integration and test thoroughly.
  </Step>
  <Step title="Document">
    Write clear setup and usage documentation.
  </Step>
  <Step title="Submit">
    Fill out our integration submission form.
  </Step>
  <Step title="Review">
    We test and review your integration.
  </Step>
  <Step title="Iterate">
    We may request changes or improvements.
  </Step>
  <Step title="Launch">
    Your integration goes live in the directory.
  </Step>
</Steps>

## Submission Requirements

### Technical
- [ ] Works with current CC API version
- [ ] Handles authentication properly
- [ ] Includes error handling
- [ ] Tested with production-like data

### Documentation
- [ ] Setup guide with screenshots
- [ ] Configuration options explained
- [ ] Troubleshooting section
- [ ] Contact for support

### Business
- [ ] Company/developer information
- [ ] Pricing model (if applicable)
- [ ] Support commitment
- [ ] Privacy policy
