## Funnel Deployment

### Deploying Behind Existing Links

If partners already have affiliate links, you can upgrade them to co-branded funnels:

<Steps>
  <Step title="Map Existing Partners">
    Identify all active partners with existing affiliate links
  </Step>
  
  <Step title="Create CC Profiles">
    Create CreatorCommerce records for each partner (can be automated)
  </Step>
  
  <Step title="Generate CC Links">
    Each partner gets a co-branded URL (e.g., `/pages/creators/partner-handle`)
  </Step>
  
  <Step title="Update Link Destinations">
    Option A: Update affiliate links to point to CC pages
    Option B: Use redirect rules to route traffic
  </Step>
  
  <Step title="Verify Tracking">
    Ensure affiliate commission tracking still works with new flow
  </Step>
</Steps>

### Link Architecture

<Tabs>
  <Tab title="Direct CC Links">
    **Format:** `yourstore.com/pages/creators/partner-handle`
    
    **Pros:**
    - Clean, branded URLs
    - Full CC experience
    - Easy to share
    
    **Cons:**
    - Requires updating existing links
    - Partner needs to know new URL
  </Tab>
  
  <Tab title="Redirect Approach">
    **Format:** Existing affiliate link â†’ CC page
    
    **Pros:**
    - No link changes needed
    - Existing tracking preserved
    - Gradual rollout possible
    
    **Cons:**
    - Extra redirect (minor speed impact)
    - More complex setup
  </Tab>
  
  <Tab title="Hybrid with UTM">
    **Format:** `yourstore.com/partner?ref=partner-handle`
    
    **Pros:**
    - Works with existing link structure
    - Query parameter triggers CC experience
    - Easy A/B testing
    
    **Cons:**
    - Less clean URLs
    - Requires detection logic
  </Tab>
</Tabs>

## Tracking Implementation

### Key Metrics to Track

<AccordionGroup>
  <Accordion title="Conversion Rate (CVR)">
    **What:** Percentage of visitors who purchase
    
    **Why it matters:** Primary measure of funnel effectiveness
    
    **Benchmark:** 30%+ CVR is the CC target (vs. typical 2-3% eComm)
    
    **How to measure:**
    - Orders attributed to partner / Sessions on partner page
    - Segment by partner, segment, and funnel variation
  </Accordion>
  
  <Accordion title="Average Order Value (AOV)">
    **What:** Average revenue per order
    
    **Why it matters:** Measures depth of purchase
    
    **Benchmark:** 67%+ higher than non-partner traffic
    
    **How to measure:**
    - Total revenue from partner / Orders from partner
    - Compare to site-wide AOV
  </Accordion>
  
  <Accordion title="Customer Lifetime Value (CLTV)">
    **What:** Total value of a customer over time
    
    **Why it matters:** Partner-acquired customers may have different retention
    
    **Benchmark:** Varies by vertical; track trend over time
    
    **How to measure:**
    - Cohort analysis by acquisition source
    - Tag customers with partner attribution at purchase
  </Accordion>
  
  <Accordion title="Affiliate Retention">
    **What:** Partner engagement and activity over time
    
    **Why it matters:** Active partners drive ongoing revenue
    
    **Benchmark:** 50%+ monthly active rate for established programs
    
    **How to measure:**
    - Partners generating traffic in period / Total partners
    - Partners generating sales in period / Total partners
  </Accordion>
</AccordionGroup>

### Attribution Implementation

<Tabs>
  <Tab title="Cart Attributes">
    Store partner info in cart for checkout attribution:
    
    ```liquid
    {% assign cc_handle = cart.attributes['cc-creator-handle'] %}
    {% assign cc_code = cart.attributes['cc-discount-code'] %}
    ```
    
    These persist to the order and enable reporting.
  </Tab>
  
  <Tab title="Order Tags">
    Tag orders with partner attribution for easy filtering:
    
    ```
    cc-partner:[handle]
    cc-segment:[segment]
    cc-discount:[code]
    ```
    
    Use Shopify Flow or API to automate tagging.
  </Tab>
  
  <Tab title="Customer Tags">
    Tag customers for CLTV tracking:
    
    ```
    cc-acquired:[handle]
    cc-first-order:[date]
    ```
    
    Enables cohort analysis by partner.
  </Tab>
</Tabs>

## Common Launch Issues

<Warning>
**Issue:** Partner pages showing blank or broken

**Cause:** Metaobject not published, template not assigned, or missing data

**Fix:** Check CC dashboard for partner status, verify template assignment
</Warning>

<Warning>
**Issue:** Discount code not applying at checkout

**Cause:** Code mismatch, expired code, or cart attribute not passed

**Fix:** Verify code in Shopify admin, check cart attribute implementation
</Warning>

<Warning>
**Issue:** Attribution not tracking

**Cause:** Cart attributes not set, order tags not firing

**Fix:** Test full funnel flow, verify Shopify Flow or API automation
</Warning>

<Warning>
**Issue:** Partner seeing wrong content

**Cause:** Wrong creator handle, cached data, or data sync issue

**Fix:** Clear caches, verify metaobject data, resync from CC
</Warning>
