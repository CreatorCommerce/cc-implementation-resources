CreatorCommerce provides custom triggers and actions in Shopify Flow.

## CC Triggers

| Trigger | Fires When | Common Use |
|---------|------------|------------|
| `CC Partner Created` | New creator added | Generate initial content, send welcome |
| `CC Partner Activated` | Partner completes activation | Enable landing page, create discount |
| `CC Partner Updated` | Any field changes | Regenerate dependent content |
| `CC Order Attributed` | Sale linked to partner | Update stats, notify partner |
| `CC Drop Updated` | Partner updates picks | Refresh landing page |
| `CC Milestone Reached` | Performance goal hit | Unlock tier, celebrate |

## CC Actions

| Action | What It Does | Common Use |
|--------|--------------|------------|
| `Update Creator Field` | Write to any custom field | Store generated/fetched data |
| `Generate Content` | AI-create text | Bios, headlines, testimonials |
| `Sync to Klaviyo` | Push data to profiles | Enable co-branded emails |
| `Create Redirect` | Set up URL forwarding | Vanity URLs |
| `Tag Order/Customer` | Add Shopify tags | Segmentation, reporting |
