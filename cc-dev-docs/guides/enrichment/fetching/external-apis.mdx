Partners have presence beyond your store. Pull it in.

## Social Platform Data

| Platform | What to Fetch | API Needed |
|----------|---------------|------------|
| Instagram | Follower count, bio, profile pic | Instagram Graph API |
| TikTok | Follower count, engagement | TikTok API |
| YouTube | Subscriber count, channel info | YouTube Data API |
| Twitter/X | Follower count, bio | X API v2 |

### Example: Sync Instagram Stats

**Trigger:** CC Creator Field Updated (cc-instagram-handle)

**Actions:**
1. HTTP Request to Instagram API
2. Parse response for followers, bio, profile pic
3. Store in custom fields
4. Assign audience tier (nano/micro/mid/macro)

```json
{
  "cc-social-followers": 45000,
  "cc-social-bio": "Fitness coach helping busy moms...",
  "cc-social-avatar": "https://instagram.com/...",
  "cc-audience-tier": "micro-influencer"
}
```

## Affiliate Platform Data

| Platform | What to Sync |
|----------|--------------|
| Impact | Historical commissions, performance |
| ShareASale | Sales history, partner tier |
| Refersion | Order data, payment history |

## CRM Data

Sync from HubSpot, Salesforce, etc:
- Customer lifetime value (if they were a customer first)
- Support ticket history
- Sales team notes
- Email engagement metrics

## Refresh Strategies

| Data Type | Frequency | Why |
|-----------|-----------|-----|
| Social followers | Weekly | Changes slowly |
| Order stats | Real-time | Accuracy matters |
| CRM data | Daily | Sync on changes |
| Performance tiers | Weekly | Stability over noise |
