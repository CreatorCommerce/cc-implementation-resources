## What You Can Do

Run consumer co-branded lifecycle messaging (B2C) by syncing creator-context events from Shopify Flow into Klaviyo.

Core event use cases:

- `Creator Checkout Abandoned`
- `Creator Attributed Order`
- `Marketing Opt In From Creator`
