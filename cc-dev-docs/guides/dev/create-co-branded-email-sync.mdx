## Klaviyo Integration

### Data Sync Flow

```
CreatorCommerce → Klaviyo Profile Properties → Flow Triggers → Co-branded Emails
```

### Syncing Creator to Klaviyo Profile

When a customer lands on a creator page, sync their creator context:

```javascript
// In your theme JavaScript
async function syncToKlaviyo(creatorData, customerEmail) {
  const klaviyoPayload = {
    token: 'YOUR_KLAVIYO_PUBLIC_KEY',
    properties: {
      $email: customerEmail,
      // CC-specific properties
      cc_creator_handle: creatorData.handle,
      cc_creator_name: creatorData.firstName + ' ' + creatorData.lastName,
      cc_discount_code: creatorData.discountCode,
      cc_discount_amount: creatorData.discountAmount,
      cc_creator_profile_photo: creatorData.profilePhoto,
      cc_landing_page_url: window.location.href,
      cc_landing_timestamp: new Date().toISOString()
    }
  };
  
  await fetch('https://a.klaviyo.com/api/identify', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(klaviyoPayload)
  });
}
```

### Klaviyo Flow Trigger

Create a flow triggered by `CC Creator Page View`:

```json
{
  "trigger": {
    "type": "metric",
    "metric_name": "CC Creator Page View"
  },
  "filters": [
    {
      "property": "$value.creator_handle",
      "operator": "is not empty"
    }
  ]
}
```

## CC-Specific Klaviyo Events

### Track Product Added from Creator Page

```javascript
function trackCreatorAddToCart(product, creator) {
  _learnq.push(['track', 'CC Add to Cart', {
    'ProductName': product.title,
    'ProductID': product.id,
    'Price': product.price,
    'CreatorHandle': creator.handle,
    'CreatorName': creator.firstName,
    'DiscountCode': creator.discountCode
  }]);
}
```

### Track Checkout Started with Creator Context

```javascript
function trackCreatorCheckoutStarted(cart, creator) {
  _learnq.push(['track', 'CC Checkout Started', {
    'ItemCount': cart.item_count,
    'TotalPrice': cart.total_price / 100,
    'CreatorHandle': creator.handle,
    'CreatorName': creator.firstName,
    'DiscountCode': creator.discountCode,
    'Items': cart.items.map(item => ({
      'ProductID': item.product_id,
      'ProductName': item.title,
      'Quantity': item.quantity,
      'Price': item.price / 100
    }))
  }]);
}
```

## Custom Email Platform Integration

### Webhook to Email Platform

```javascript
// Express webhook handler
router.post('/webhooks/cc-to-email', async (req, res) => {
  const { event, data } = req.body;
  
  switch (event) {
    case 'collab.created':
      // Sync new partner to email platform
      await emailPlatform.upsertContact({
        email: data.creator.email,
        tags: ['cc-partner'],
        customFields: {
          cc_creator_handle: data.creator.handle,
          cc_tier: data.collab.tier
        }
      });
      break;
      
    case 'order.attributed':
      // Update customer profile with creator context
      await emailPlatform.upsertContact({
        email: data.customerEmail,
        customFields: {
          cc_last_creator_handle: data.creatorHandle,
          cc_last_creator_name: data.creatorName,
          cc_total_orders: data.totalOrders
        }
      });
      break;
  }
  
  res.status(200).send('OK');
});
```

### Mailchimp Merge Tags

```javascript
async function syncToMailchimp(customer, creator) {
  const mailchimp = require('@mailchimp/mailchimp_marketing');
  
  mailchimp.setConfig({
    apiKey: process.env.MAILCHIMP_API_KEY,
    server: process.env.MAILCHIMP_SERVER_PREFIX
  });
  
  await mailchimp.lists.setListMember(
    process.env.MAILCHIMP_LIST_ID,
    crypto.createHash('md5').update(customer.email.toLowerCase()).digest('hex'),
    {
      email_address: customer.email,
      merge_fields: {
        FNAME: customer.firstName,
        CC_CREATOR: creator.firstName,
        CC_HANDLE: creator.handle,
        CC_DISCOUNT: creator.discountCode,
        CC_AMOUNT: creator.discountAmount.toString()
      },
      tags: ['cc-customer', `cc-${creator.tier}`]
    }
  );
}
```

## Testing & Debugging

### Verify Sync is Working

```javascript
// In browser console on creator page
console.log('Klaviyo queue:', window._learnq);

// Check if identify was called
fetch('https://a.klaviyo.com/api/v2/list/LIST_ID/members?api_key=YOUR_KEY')
  .then(r => r.json())
  .then(data => console.log('Members:', data));
```

### Preview Dynamic Content

In Klaviyo:
1. Go to Templates → Edit template
2. Click "Preview" 
3. Select a profile with `cc_creator_handle` populated
4. Verify dynamic content renders correctly
