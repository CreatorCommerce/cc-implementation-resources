## Available CC Triggers

| Trigger | Fires When |
|---------|------------|
| `CC Order Attributed` | An order is attributed to a partner |
| `CC Partner Created` | A new partner/collab is created |
| `CC Partner Activated` | A partner activates their page |
| `CC Tier Changed` | A partner's tier changes |
| `CC Drop Created` | A partner creates a new drop |
| `CC Partner Milestone` | Partner hits a custom milestone |

## Building Flows in Shopify Admin

### Step 1: Access Shopify Flow

1. Go to Shopify Admin â†’ Apps â†’ Flow
2. Click "Create workflow"
3. Choose a trigger

### Step 2: Select CC Trigger

Search for "CreatorCommerce" in the trigger list:

1. Select your desired trigger (e.g., "CC Order Attributed")
2. Configure any trigger filters if needed

### Step 3: Add Conditions

Click "+" â†’ "Condition" to add logic:

```
If partner.tier equals "starter"
AND attribution.orderValue >= 100
```

### Step 4: Add Actions

Click "+" â†’ "Action" to add operations:

1. Search for "CreatorCommerce" actions
2. Configure action parameters
3. Use dynamic values from trigger data

## Custom Flow Actions via HTTP

For actions not built-in, use the HTTP Request action:

### Update External CRM

```yaml
action:
  type: send_http_request
  method: POST
  url: https://api.hubspot.com/contacts/v1/contact/email/{{ partner.email }}/profile
  headers:
    Authorization: Bearer {{ secrets.HUBSPOT_API_KEY }}
    Content-Type: application/json
  body: |
    {
      "properties": [
        { "property": "cc_tier", "value": "{{ partner.tier }}" },
        { "property": "cc_total_orders", "value": "{{ partner.metrics.totalOrders }}" }
      ]
    }
```

### Trigger Zapier Webhook

```yaml
action:
  type: send_http_request
  method: POST
  url: https://hooks.zapier.com/hooks/catch/123456/abcdef/
  body: |
    {
      "event": "partner_milestone",
      "partner_email": "{{ partner.email }}",
      "milestone": "first_order",
      "order_value": {{ attribution.orderValue }}
    }
```

## Flow Templates for CC

### Tiered Commission Structure

```yaml
# Automatically adjust commission based on performance
trigger: cc_partner_milestone
conditions:
  - milestone.type: "monthly_revenue"
  
branches:
  - if: milestone.value >= 10000
    actions:
      - type: cc_update_partner_field
        field: commission_rate
        value: 0.25
      - type: cc_update_partner_tier
        tier: platinum
        
  - if: milestone.value >= 5000
    actions:
      - type: cc_update_partner_field
        field: commission_rate
        value: 0.20
      - type: cc_update_partner_tier
        tier: gold
        
  - if: milestone.value >= 1000
    actions:
      - type: cc_update_partner_field
        field: commission_rate
        value: 0.15
      - type: cc_update_partner_tier
        tier: silver
```

### Dormant Partner Re-engagement

```yaml
trigger: scheduled
schedule: weekly
  
query:
  type: cc_partners
  conditions:
    - tier: not_equal "inactive"
    - last_order_date: older_than "30 days"
    
for_each: partner
actions:
  - type: cc_send_partner_email
    template: reengagement_nudge
    
  - wait: 14 days
  
  - condition:
      partner.orders_in_last_30_days: 0
    action:
      type: cc_update_partner_tier
      tier: inactive
```

## Debugging Flows

### View Flow Run History

1. Go to Shopify Admin â†’ Apps â†’ Flow
2. Select your workflow
3. Click "Runs" tab
4. Inspect individual run details

### Common Issues

| Issue | Solution |
|-------|----------|
| Trigger not firing | Verify CC webhook is configured |
| Condition always false | Check data types (string vs number) |
| Action failing | Check API credentials and permissions |
| Email not sending | Verify partner email is valid |
