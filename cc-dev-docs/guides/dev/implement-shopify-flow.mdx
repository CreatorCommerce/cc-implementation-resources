## What You Can Do

Use Shopify Flow to automate creator/collab operations and downstream system updates.

Common automation outcomes:

- Sync collab/tier updates into CreatorCommerce
- React to creator state changes (handle/code updates)
- Trigger CRM, ESP, Slack, and HTTP workflows
- Power consumer co-branded lifecycle events in Klaviyo
