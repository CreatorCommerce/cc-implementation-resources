## What You Can Do

Request API access to build custom data integrations, run enrichment workflows, and connect external systems to CreatorCommerce.

## How To Get Access

Email the team at [help@creatorcommerce.shop](mailto:help@creatorcommerce.shop) with:

- Your Shopify store domain
- Primary technical contact
- Intended use case (for example: reporting, automation, CRM sync, custom storefront logic)

The team will share access details and next steps.
