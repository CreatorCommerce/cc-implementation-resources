## What You Can Do

Extend co-branded commerce experiences to external publisher/affiliate surfaces with embeddable components, while preserving attribution context and conversion continuity.

Common use cases:

- Embedded product modules on creator-owned sites
- Campaign-specific off-store buying surfaces
- Partner microsites with creator-context merchandising
