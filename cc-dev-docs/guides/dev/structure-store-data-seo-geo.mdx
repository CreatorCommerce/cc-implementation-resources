## Overview

Co-branded creator pages can rank for:

- Creator name searches ("Sarah Jones skincare routine")
- Product + endorsement searches ("best moisturizer dermatologist recommended")
- Local searches (if creator has geographic relevance)
- AI/GEO search (ChatGPT, Perplexity, etc.)

This guide covers technical SEO and emerging GEO optimization.

## Schema.org Structured Data

### Person Schema for Creator

```liquid
{% if cc_creator %}
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "{{ cc_creator.cc-creator-first-name }} {{ cc_creator.cc-creator-last-name }}",
  "description": {{ cc_creator.cc-creator-shop-description | json }},
  "image": "{{ cc_creator.data.value.cc-creator-profile-picture | image_url: width: 400 }}",
  "url": "{{ shop.url }}/pages/creators/{{ cc_creator.handle }}",
  {% if cc_creator.data.value.instagram %}
  "sameAs": [
    "https://instagram.com/{{ cc_creator.data.value.instagram | remove: '@' }}"
    {% if cc_creator.data.value.tiktok %},
    "https://tiktok.com/@{{ cc_creator.data.value.tiktok | remove: '@' }}"
    {% endif %}
  ],
  {% endif %}
  "jobTitle": {{ cc_creator.data.value.profession | default: 'Content Creator' | json }},
  "affiliation": {
    "@type": "Organization",
    "name": "{{ shop.name }}",
    "url": "{{ shop.url }}"
  }
}
</script>
{% endif %}
```

### ItemList for Creator's Product Picks

```liquid
{% if cc_creator and cc_creator.cc-creator-drops.value %}
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "ItemList",
  "name": "{{ cc_creator.cc-creator-first-name }}'s Product Picks",
  "description": "Curated products recommended by {{ cc_creator.cc-creator-first-name }} {{ cc_creator.cc-creator-last-name }}",
  "itemListElement": [
    {% assign drops_data = cc_creator.data.value.cc-creator-drops %}
    {% for drop in drops_data limit: 10 %}
      {% assign collection_id = drop['cc-creator-drop-collection-id'] %}
      {% for collection in collections %}
        {% if collection.id | append: '' == collection_id %}
          {% for product in collection.products limit: 5 %}
          {
            "@type": "ListItem",
            "position": {{ forloop.index }},
            "item": {
              "@type": "Product",
              "name": {{ product.title | json }},
              "description": {{ product.description | strip_html | truncate: 200 | json }},
              "image": "{{ product.featured_image | image_url: width: 800 }}",
              "url": "{{ shop.url }}{{ product.url }}",
              "offers": {
                "@type": "Offer",
                "price": "{{ product.price | money_without_currency }}",
                "priceCurrency": "{{ cart.currency.iso_code }}",
                "availability": "{% if product.available %}https://schema.org/InStock{% else %}https://schema.org/OutOfStock{% endif %}"
              }
            }
          }{% unless forloop.last %},{% endunless %}
          {% endfor %}
        {% endif %}
      {% endfor %}
    {% endfor %}
  ]
}
</script>
{% endif %}
```

## Content Optimization for GEO

### What is GEO?

Generative Engine Optimization (GEO) optimizes content for AI-powered search engines like ChatGPT, Perplexity, Google AI Overviews, and Bing Copilot.

### Key GEO Principles

1. **Clear, direct answers** — Structure content as Q&A
2. **Authoritative attribution** — Cite credentials and expertise
3. **Structured data** — Help AI understand relationships
4. **Concise summaries** — Provide scannable key points

### GEO-Optimized Creator Bio

```liquid
<section class="creator-bio" itemscope itemtype="https://schema.org/Person">
  <h1 itemprop="name">{{ cc_creator.cc-creator-first-name }} {{ cc_creator.cc-creator-last-name }}</h1>
  
  {% if cc_creator.data.value.profession %}
    <p class="creator-title" itemprop="jobTitle">
      {{ cc_creator.data.value.profession }}
    </p>
  {% endif %}
  
  {% if cc_creator.data.value.credentials %}
    <p class="creator-credentials">
      <strong>Credentials:</strong> {{ cc_creator.data.value.credentials }}
    </p>
  {% endif %}
  
  <div class="creator-summary" itemprop="description">
    {{ cc_creator.cc-creator-shop-description }}
  </div>
  
  {% if cc_creator.data.value.expertise %}
    <div class="creator-expertise">
      <strong>Areas of Expertise:</strong>
      <ul>
        {% assign expertise_list = cc_creator.data.value.expertise | split: ',' %}
        {% for area in expertise_list %}
          <li>{{ area | strip }}</li>
        {% endfor %}
      </ul>
    </div>
  {% endif %}
</section>
```

### FAQ Schema for AI Search

```liquid
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "Who is {{ cc_creator.cc-creator-first-name }} {{ cc_creator.cc-creator-last-name }}?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "{{ cc_creator.cc-creator-first-name }} is a {{ cc_creator.data.value.profession | default: 'content creator' }} who partners with {{ shop.name }}. {{ cc_creator.cc-creator-shop-description | truncate: 200 }}"
      }
    },
    {
      "@type": "Question",
      "name": "What products does {{ cc_creator.cc-creator-first-name }} recommend?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "{{ cc_creator.cc-creator-first-name }} has curated a selection of their favorite products from {{ shop.name }}, including {% for drop in cc_creator.data.value.cc-creator-drops limit: 3 %}{{ drop['cc-creator-drop-name'] }}{% unless forloop.last %}, {% endunless %}{% endfor %}."
      }
    },
    {
      "@type": "Question",
      "name": "How do I get {{ cc_creator.cc-creator-first-name }}'s discount?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Use code {{ cc_creator.cc-collab-discount-code }} at checkout to get {{ cc_creator.cc-collab-discount-amount }}% off your order."
      }
    }
  ]
}
</script>
```

## Sitemap Configuration

### Include Creator Pages

```xml
<!-- In sitemap.xml -->
{% for creator in metaobjects.creator %}
  <url>
    <loc>{{ shop.url }}/creators/{{ creator.handle }}</loc>
    <lastmod>{{ creator.updated_at | date: '%Y-%m-%d' }}</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.7</priority>
  </url>
{% endfor %}
```

### Robots.txt Considerations

```
# Allow creator pages
Allow: /creators/
Allow: /pages/creators/

# Block duplicate parameter URLs
Disallow: /*?ref=*
Disallow: /*?utm_*
```
