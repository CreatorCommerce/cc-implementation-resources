## What You Can Do

Create branded, workflow-specific creator experiences when native forms/dashboard settings are not enough.

Common use cases:

- Custom application/onboarding workflows
- Campaign-specific data collection flows
- Embedded internal dashboards for ops teams
- Specialized creator portals with external system integration
