## What's Covered

### Funnel A: Out-of-the-Box Implementation

- Fastest path to launch using CC templates.
- Best for early validation and fast iteration.

### Funnel B: Theme Conversion Implementation

- Retrofit existing storefront UI with creator context.
- Best when you want to preserve current design system and page structure.

### Funnel C: Net-New Funnel Build

- Build a fully custom creator funnel architecture from first principles.
- Includes mobile-first, merchandising, offer, checkout, messaging, and performance best-practice checklist.
- Best for high-intent paid/affiliate traffic and premium co-branded experiences.
