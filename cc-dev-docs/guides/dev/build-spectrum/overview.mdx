CreatorCommerce offers a spectrum of build approaches. Start with zero-code hosted solutions and progressively customize toward fully white-labeled platforms—all running on the same CC infrastructure.

```
│                                                                          │
│  OUT-OF-THE-BOX                                           WHITE-LABEL   │
│  CC Hosted                                                Your Platform │
│                                                                          │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌────────┐│
│  │  App     │   │  Theme   │   │  Custom  │   │ Headless │   │ Full   ││

│  Zero code      Low code       Medium        High code     Full control │

```

## Comparison

| Approach | Time to Launch | Customization | Maintenance | Best For |
|----------|---------------|---------------|-------------|----------|
| App Blocks | Minutes | Low | Zero | Quick start, testing |
| Theme Sections | Hours | Medium | Low | Brand consistency |
| Custom CSS | Days | Medium-High | Low | Design-forward brands |
| Headless API | Weeks | High | Medium | Custom experiences |
| White-Label | Months | Complete | Higher | Platform businesses |

## What Stays the Same

Regardless of where you are on the spectrum, you always get:

- **CC Data Model** — Creator, collab, drop data structure
- **Shopify-Native Storage** — Metaobjects in your store
- **Attribution Engine** — Order tracking and partner credit
- **Integration Layer** — Klaviyo, Flow, analytics sync
- **Dashboard & Forms** — Partner self-service (customizable)

## What Changes

As you move toward white-label:

| Moving Right → | You Gain | You Take On |
|----------------|----------|-------------|
| Theme Sections | Design control | Section maintenance |
| Custom CSS | Brand expression | CSS management |
| Headless API | Complete UI freedom | Frontend development |
| White-Label | Platform ownership | Full stack responsibility |

## Progressive Customization

You don't have to choose once and forever. Start simple, customize progressively:

1. **Launch with App Blocks** — Get live fast
2. **Add Theme Sections** — For key landing pages
3. **Apply Custom CSS** — Match brand guidelines
4. **Build Custom Features** — Via API as needed
5. **Go Full White-Label** — If business requires
