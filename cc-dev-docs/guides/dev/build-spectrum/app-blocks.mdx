App Blocks are the fastest path to co-branded experiences. Drag and drop CC components into your theme editor—no code required.

## What You Get

CC provides ready-to-use app blocks for:

| Block | Function |
|-------|----------|
| Creator Hero | Partner intro with photo, bio, social |
| Product Grid | Partner's curated drops |
| Discount Banner | Partner's discount code display |
| Trust Bar | Partner endorsement strip |
| Testimonial | Partner quote or message |

## How It Works

```
┌─────────────────────────────────────────────┐
│           Shopify Theme Editor              │
├─────────────────────────────────────────────┤
│                                             │
│  ┌─────────────────────────────────────┐   │
│  │         Your Theme Section          │   │
│  └─────────────────────────────────────┘   │
│                                             │
│  ┌─────────────────────────────────────┐   │

│  └─────────────────────────────────────┘   │
│                                             │
│  ┌─────────────────────────────────────┐   │
│  │    + Add Block → CC Product Grid    │   │

│                                             │
└─────────────────────────────────────────────┘
```

## Setup

<Steps>
  <Step title="Install CC App">
    Install CreatorCommerce from the Shopify App Store.
  </Step>
  <Step title="Open Theme Editor">
    Customize your theme in Shopify admin.
  </Step>
  <Step title="Add Blocks">
    In any section that supports blocks, add CC components.
  </Step>
  <Step title="Configure">
    Adjust block settings (colors, layout, content shown).
  </Step>
  <Step title="Save">
    Publish and you're live.
  </Step>
</Steps>

## Customization Options

Each block includes settings for:

- **Layout** — Alignment, spacing, columns
- **Content** — Which fields to show/hide
- **Colors** — Override defaults (or inherit from creator)
- **Typography** — Font size adjustments
- **Responsive** — Mobile-specific settings

## Limitations

App Blocks are quick but constrained:

| ✅ Can Do | ❌ Can't Do |
|-----------|-------------|
| Add to any page | Deep layout changes |
| Adjust settings | Custom HTML structure |
| Change colors | Complex animations |
| Show/hide fields | Unique per-page designs |

## When to Use

**Use App Blocks when:**
- You're testing CC before deeper integration
- You need co-branding on a few pages quickly
- Your theme already has the structure you want
- Non-technical team needs to make changes

**Move beyond App Blocks when:**
- You need pixel-perfect brand matching
- You want unique layouts per partner type
- You're building dedicated landing pages
- You need custom functionality
