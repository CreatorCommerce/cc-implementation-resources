## The Data Flow

```
Customer lands on creator page
         ↓
CC captures creator context
         ↓
Context synced to email platform
         ↓
Profile enriched with creator data
         ↓
Flows triggered with personalization
         ↓
Co-branded emails sent
```

## What Gets Synced

| Data Point | Source | Destination |
|------------|--------|-------------|
| Creator handle | Cart attributes | Profile property |
| Creator name | CC metaobject | Profile property |
| Discount code | Collab data | Profile property |
| Discount amount | Collab data | Profile property |
| Profile photo URL | Creator data | Profile property |
| Landing page URL | Session | Profile property |

## Sync Triggers

| Event | What Happens |
|-------|--------------|
| **Page view** | Creator context captured |
| **Identify** | Profile created/updated with CC data |
| **Add to cart** | Event includes creator context |
| **Checkout** | Full attribution passed |
| **Purchase** | Order linked to creator |

## Using in Emails

Access these properties in your email templates for personalization. Each platform has its own syntax—see platform-specific guides.
