## The Approach

Don't create separate flows for creator traffic. Add conditional blocks to existing flows.

## Flow-by-Flow Guide

### Welcome Series

**Email 1: Welcome**
```html
{% if person.cc_creator_name %}
  <p>Welcome! You found us through {{ person.cc_creator_name }}.</p>
  <p>Your exclusive code: <strong>{{ person.cc_discount_code }}</strong></p>
{% else %}
  <p>Welcome to {{ organization.name }}!</p>
{% endif %}
```

### Cart Abandonment

**Email 1: Reminder (4 hours)**
```html
{% if person.cc_creator_name %}
  <h2>{{ person.cc_creator_name }} picked these for you</h2>
  <p>Don't forget your {{ person.cc_discount_amount }}% off!</p>
{% else %}
  <h2>You left something behind</h2>
{% endif %}
```

### Post-Purchase

**Email 1: Thank You**
```html
{% if person.cc_creator_name %}
  <div class="creator-thanks">
    <img src="{{ person.cc_creator_image }}" />
    <p>Thanks for shopping with {{ person.cc_creator_name }}!</p>
  </div>
{% endif %}
```

### Browse Abandonment

```html
{% if person.cc_creator_name %}
  <p>{{ person.cc_creator_name }} thinks you'd love these:</p>
{% else %}
  <p>Still thinking about these?</p>
{% endif %}
```

### Win-Back

```html
{% if person.cc_creator_name %}
  <h2>{{ person.cc_creator_name }} has new picks for you</h2>
  <p>Your code {{ person.cc_discount_code }} is still active!</p>
{% else %}
  <h2>We miss you!</h2>
{% endif %}
```

## Block Placement

Add creator blocks at high-impact positions:
1. **Hero area** — First thing they see
2. **Before CTA** — Reinforce the discount
3. **Footer** — Reminder of the relationship

## Keep Generic Fallbacks

Every co-branded block needs a fallback. Not all customers will have creator context.
