# CreatorCommerce Developer Platform

> Updates to the CreatorCommerce Developer Platform, including the Unified API, client SDKs, and developer tools.

<Tip>
  For release notes on other CreatorCommerce products, see the [CreatorCommerce Help Center](https://support.creatorcommerce.com).
</Tip>

#### April 23, 2026

* **Merchant operations guidance**: Added clearer merchant workflows for creator invite links, storefront testing, safe redirects, forms, dashboard metrics, and integration setup. See [What You Can Build](/use-cases/what-you-can-build).

#### April 20, 2026

* **New ShopifyQL Reference**: Added a ShopifyQL reference page for analyzing CreatorCommerce-tagged traffic in Shopify, including conversion funnel and AOV example queries. See [ShopifyQL](/references/shopifyql).

#### February 21, 2026

* **Docs MCP Launch**: We launched the CreatorCommerce documentation MCP endpoint and setup guide for AI-assisted development. Start with [Connect to CreatorCommerce MCP](/guides/mcp).

* **AI Setup Docs Refresh**: Cursor and Claude setup pages now point to the MCP-first flow so setup is consistent across coding assistants. See [Coding in Cursor](/guides/cursor-coding) and [Coding in Claude Code](/guides/claude-coding).

* **New Codex Setup Resource**: Added a dedicated Codex setup page that uses the same MCP-first pattern. See [Coding in Codex](/guides/codex-coding).

* **MCP Prompting Resources**: Added/updated practical MCP usage references for implementation workflows. See [MCP Examples](/guides/mcp-examples).
